# AnimeTimeRecorder 用户手册

## 1. 这是什么
AnimeTimeRecorder 是一个**离线优先**的动画/追番记录工具（PWA）。

- 无账号、无服务器：你的记录保存在本机（浏览器的 IndexedDB）
- 在线能力：联网时检索条目并补全信息
- 离线能力：可离线打开、离线查看已缓存内容、离线本地搜索

适用人群：想要一个“像 App 一样”的追番记录工具，并希望在断网时也能查到自己常用条目与关联信息。

---

## 2. 功能与特点

### 2.1 记录与管理
- 状态：
  - 想看（wish）
  - 在看（doing）
  - 看完（done）
  - 搁置（on_hold）
  - 弃坑（dropped）
- 评分：0-10（0.5 步进）；填 0 表示不评分
- 自定义中文标题：用于覆盖显示标题（当默认中文名不合你意）

### 2.2 离线缓存（打开条目自动补全）
打开条目弹窗后，会在联网时自动后台缓存：
- 条目详情（简介、放送日期、平台、评分等）
- 条目封面（离线可看）
- 条目扩展信息（人物/制作人员、角色、声优）

离线时打开条目：
- 如果本地有缓存，会显示缓存内容
- 如果本地无缓存，会提示需要联网

### 2.3 离线搜索增强
离线/本地搜索不仅匹配标题，还会匹配：
- 条目别名（aliases）
- 人物/制作人员
- 角色
- 声优

### 2.4 搜索历史
搜索页会保存最近 5 条搜索关键词，便于快速复用。

### 2.5 批量操作（我的记录页）
进入批量模式后，可对多个条目进行：
- 批量改状态
- 批量清评分
- 批量设评分
- 批量删除记录（仅删除记录，不会自动删除缓存；可在设置页手动清理缓存）

进入/退出方式：
- 手机：长按卡片约 0.5s 进入批量并自动选中
- 桌面：Shift+点击 或 右键卡片 进入批量并自动选中
- 退出：点击“退出”按钮、点击页面标题“我的记录”、或点击页面空白处

### 2.6 PWA：安装与更新提示
- 支持“添加到主屏幕”，体验更像独立 App
- 有更新时会提示刷新更新
- 断网时会提示离线状态

说明：
- 并非所有浏览器都支持 PWA 的“一键安装”。
- 若浏览器支持安装，应用顶部会出现“安装”按钮（系统会弹出确认框）。
- 若浏览器不支持安装按钮，你仍可使用浏览器菜单里的“添加到桌面/添加到主屏幕/创建快捷方式”。
- 想要最稳定的 PWA 安装体验，建议使用 Chrome/Edge。

### 2.7 外观：背景图
- 默认背景：`Kirigiri.png`
- 你也可以在设置页导入自定义背景图
  - 自定义背景会覆盖默认背景
  - 清除背景图后会恢复默认背景

## 3. 部署与使用（给普通用户）

你有两种常见使用方式：

### 3.1 方式 A（推荐）：公网链接部署（Cloudflare Pages）
适合：发给别人用、跨设备访问、HTTPS 下 PWA 安装更稳定。

**部署者操作（一次性）：**
1. 将项目推送到 GitHub
2. Cloudflare Pages 绑定 GitHub 仓库
3. 设定构建参数：
   - Framework preset：Vite
   - Root directory：`app`
   - Build command：`npm run build`
   - Build output directory：`dist`
4. 发布后会得到一个 `https://xxx.pages.dev` 链接

**使用者操作：**
1. 打开链接
2. 在浏览器菜单中选择“安装应用 / 添加到主屏幕”

### 3.2 方式 B：局域网测试（开发预览）
适合：开发测试、同一 Wi‑Fi 环境下手机访问。

**电脑端：**
1. 进入 `app` 目录
2. 启动预览：
   - `npm run build`
   - `npm run preview -- --host`
3. 终端会输出 Network 地址，例如：`http://<电脑局域网IP>:4173/`

**手机端：**
1. 手机与电脑在同一 Wi‑Fi
2. 用手机浏览器打开 Network 地址
3. 选择“安装应用 / 添加到主屏幕”

## 4. 离线使用建议

### 4.1 如何让离线更好用
- 常用条目：建议你**至少打开一次条目详情**，让应用自动缓存封面/人物/角色
- 之后即使断网，也能在“我的记录/搜索页”通过本地缓存继续使用

### 4.2 缓存策略（可配置）
设置页提供“条目详情缓存有效期”选项：
- 1 天 / 7 天（默认）/ 30 天
- 0：永不过期（不自动刷新）

该策略会影响“打开条目时是否需要重新拉取并更新缓存”。

## 5. 数据备份与迁移

### 5.1 导出
设置页支持：
- 导出 JSON（完整备份）
- 导出 CSV（表格查看）

### 5.2 导入
- 导入 JSON 会覆盖当前本地数据（用于恢复备份/换机迁移）
- 涉及覆盖/清空/清理等危险操作，都会要求二次确认

---

## 6. 缓存清理与空间管理

设置页提供：
- 清理缓存（保留记录）：清空条目缓存与人物/角色缓存
- 仅清理封面缓存：只清封面 Blob（节省空间）
- 仅清理条目详情缓存（保留封面）
- 仅清理人物/角色缓存

## 7. 常见问题（FAQ）

### 7.1 Android + Edge：点“添加到手机 / Create shortcut”没反应
常见原因：系统桌面设置拦截了创建桌面快捷方式。

解决：在手机 **桌面/主屏幕设置** 中关闭：
- “禁止添加快捷方式 / 禁止添加到桌面”（或类似开关）
然后再尝试添加。

### 7.2 我能不能拿到 APK 安装包？
本项目是 PWA（网页应用），默认交付方式是“链接 + 安装到主屏幕”。

如确实需要 APK，需要额外进行 TWA/Capacitor 等封装与签名流程（属于后续增强）。

### 7.3 数据会不会丢？
数据存放在浏览器本地数据库中：
- 清理浏览器数据/卸载浏览器/重装系统可能导致数据丢失
- 建议定期导出 JSON 作为备份
